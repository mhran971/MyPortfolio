# Mahran Abo Dakka - Personal Resume & Portfolio

## Project Description
This is the personal resume and portfolio website of Mahran Abo Dakka, a passionate software engineering student and freelancer based in Damascus, Syria. The website serves as a professional showcase of his skills, experience, projects, and services in web development. Built on the "MyResume" template from BootstrapMade, it has been customized to reflect his personal brand and expertise.

The site highlights his proficiency in modern web technologies, including PHP, Laravel, React, and more, while demonstrating his commitment to creating innovative software solutions that solve real-world problems.

## Features
- **Hero Section**: Eye-catching introduction with animated typing effect showcasing roles like "Software Engineer", "Web Developer", "Problem Solver", and "Freelancer". Includes social media links.
- **About Section**: Personal details, including birthday, phone, city, age, degree, email, and freelance availability. Features a profile image and a brief bio.
- **Skills Section**: Divided into Frontend Development (HTML, CSS, JS, React, etc.), Backend Development (PHP, Laravel, GraphQL), and Tools & Technologies (Git, NPM, XAMPP, etc.). Interactive icons with hover effects.
- **Resume Section**: Summary, education (Bachelor's in Software Engineering, expected 2026), professional experience (3 years freelancing), and skills overview.
- **Portfolio Section**: Link to GitHub repositories for viewing all projects.
- **Services Section**: Offers web development, mobile development, database management, version control, communication, and problem-solving services.
- **Contact Section**: Contact information (location, phone, email, LinkedIn, GitHub) and a contact form (currently disabled with a note).
- **Footer**: Social links, copyright, and inspirational quotes in English and Arabic.
- **Responsive Design**: Optimized for all devices using Bootstrap.
- **Animations**: Powered by AOS for smooth scroll animations.

## Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript, React.js, Bootstrap 5, Typed.js, AOS
- **Backend**: PHP, Laravel, GraphQL, SQL
- **Tools**: Git, GitHub, NPM, Vite, XAMPP, Linux
- **Icons**: Bootstrap Icons, DevIcons
- **Fonts**: Google Fonts (Roboto, Poppins, Raleway)

## Installation/Setup
1. Download or clone the project repository to your local machine.
2. Ensure you have a web browser installed (e.g., Chrome, Firefox).
3. Open the `index.html` file directly in your browser. No server setup is required as this is a static website.
4. For development or customization, you can use any code editor like VS Code.

## Usage
- Navigate the site using the fixed navigation menu on the left.
- Scroll through sections or click menu items for smooth scrolling.
- View portfolio projects by clicking the "View All Projects on GitHub" button.
- Contact via the form (note: form submission is currently disabled; use provided contact details instead).
- The site is fully responsive; test on mobile devices for best experience.

## Project Structure
- `index.html`: Main HTML file containing the entire website structure.
- `assets/css/main.css`: Custom CSS styles.
- `assets/js/main.js`: Custom JavaScript for interactions.
- `assets/img/`: Images used in the site (profile, portfolio, etc.).
- `assets/vendor/`: Third-party libraries (Bootstrap, AOS, etc.).
- `forms/`: Contact form handler (PHP).
